## Base template for NJ DEP R Shiny Apps

### Download and extract the repository files:
- Under the **Code** tab, click the **Code** dropdown
- Click **Download ZIP**
- Extract the the **.zip** file into your project directory

### Environment setup:
(**Note:** Only follow the steps below if R doesn't automatically setup the environment for you)
- Install **renv**: `install.packages("renv")`
- Create environment: `renv::restore()`

#### See more in the [docs](https://dakotawnj.github.io/shiny-template-r/)
